Для инициализации репозитория пишем: git init . (точка - текущая папка, можно указывать полный путь).
Чтобы понять текущее состояние репозитория пишем: git status
Чтобы добавить файлы в staging area пишем: git add "имя файла или . (добавятся все измененные файлы из текущей папки)".
Чтобы сделать commit пишем: git commit -m "Описание коммита" (можно закоммитить и без -m).
Чтобы посмотреть историю коммитов пишем: git log
Чтобы перейти к конкретному коммиту пишем: git checkout "hash коммита, к которому переходим"
